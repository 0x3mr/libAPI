libAPI

API for managing books in a library
CSAI-203 Course Assignment by Amr Abdelfattah

Endpoints

POST

/books

Add a new book to the library. Requires the following fields:

  • title (string)
  • author (string)
  • publishedYear (integer)
  • isbn (string)
  • genre (string, optional)

GET

/books

Retrieve a list of all books. Optional query parameters:

  • author (string)
  • publishedYear (integer)
  • genre (string)

DELETE

/books/{isbn}

Delete a book by its ISBN.

PUT

/books/{isbn}

Update the details of a book by its ISBN. Accepts the following fields:

  • title (string)
  • author (string)
  • publishedYear (integer)
  • genre (string, optional)
